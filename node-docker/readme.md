# NODE-DOCKER - Kester Simm 
kester.simm@icloud.com

### Installation
Run node-docker-build.cmd:
```sh
$ ./node-docker-build
```
### Browse
Pattern:
http://localhost:8887
```sh
$ curl http://localhost:8887
Hello, Node is running on Kubernetes and Docker
```
### Remove from kubernetes 
Run node-docker-remove.cmd:
```sh
$ /node-docker-remove
```
