# GO-DOCKER - Kester Simm 
kester.simm@icloud.com

### Installation
Run go-docker-build.cmd:
```sh
$ ./go-docker-build
```
### Browse
Pattern:
http://localhost:8886
```sh
$ curl http://localhost:8886
Hello, Guest
```
### Remove from kubernetes 
Run go-docker-remove.cmd:
```sh
$ /go-docker-remove
```
