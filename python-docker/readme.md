### Installation
Run python-docker-build.cmd:
```sh
$ ./python-docker-build
```
### Browse
Pattern:
http://localhost:8888
```sh
$ curl http://localhost:8888
Hello, Python is running on Kubernetes and Docker
```
### Remove from kubernetes 
Run python-docker-remove.cmd:
```sh
$ /python-docker-remove
```
